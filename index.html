ℹ️ Ollama Setup

This app uses Ollama for a fully local AI assistant. To get started:

  1. Install Ollama from ollama.com
  2. Run ollama pull qwen3:4b in your terminal
  3. Make sure Ollama is running (it starts automatically after install)
  4. Click "Test Connection" above to verify

If Ollama is not running, the assistant will use built-in fallback responses.
